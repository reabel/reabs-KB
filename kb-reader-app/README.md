# KB Reader App

This application is designed to provide a frontend UI for the [KB Reader API](../kb-reader-api/readme.md)

## Monorepo

This package is part of the **reabs-kb** monorepo. Install all dependencies from the repository root:

```bash
npm install
```

Run the dev server from the root:

```bash
npm run app
```

Or from within this directory:

```bash
npm run dev
```
